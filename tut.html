Tutoriais dos HTML:

HTML é a base para construir qualquer página na web! Se você está começando ou quer relembrar os conceitos, aqui é o lugar perfeito para aprender de forma simples e prática. Use, explore e crie sem medo!

Strong

Usado para grifar (negrito) alguma palavra ou texto.

<p>Esse aviso é <strong>muito importante</strong> </p>

Nav

Usado para criar navegações no canto superior da tela.

<nav>
<a href="#inicio">Início</a>
<a href="#contato">Contato</a>
</nav>

Li (UL e OL)

Usado para criar lista ordenada (ol) ou não ordenada (ul).

<ul>
<li>Item 1</li>
<li>Item 2</li>
</ul>

Img

Usado para colocar fotos.

<img src=" caminho da imagem" alt=" texto alternativo caso a imagem não carregue">

Footer

Para colocar o copy right.

<footer>
<p> ©2025 Empresa tal</p>
</footer>

Div e Class

Criar blocos genéricos para agrupar outros elementos.

<div class="container">
<p>Texto dentro do bloco</p>
</div>

Button

Criar um botão.

<button>Clique aqui</button>

Br

Pular linha.

Linha 1
<br> Linha 2

Tutoriais dos CSS:

Quer aprender CSS de forma fácil? Aqui você vai descobrir como estilizar suas páginas, deixar tudo mais bonito e personalizado. Vamos juntos transformar seu site em algo incrível!

Border-radius

Arredondar os cantos de um elemento.

.foto{
border-radius: 15px;
}

Justify content

Alinha itens horizontalmente em um flex container.

.container{
display: flex;
justify-content: center;
}

Display

Controla o comportamento do layout.

.container{
display: flex;
}

Width e Height

Widht: Define quão largo será o elemento.

Height: Define quão alto será o elemento.

.div{
width: 100px;
height: 100px;
}

Color

Define a cor do texto.

.p{
color: red;
}

Font-size

Define o tamanho da fonte.

.h1{
font-size: 30px;
}

Background-color

Define a cor de fundo de um elemento.

.div{
background-color: red;
}

Margin

Espaço externo ao redor do elemento.

.div{
margin: 20px;
}

Padding

Espaço interno entre o conteúdo e a borda.

.div{
padding: 15px;
}

Border

Adiciona uma borda ao redor do elemento.

.div{
border: 2px solid red;
}

Text-aligh

Alinha o texto (esquerda, direita, centro).

.p{
text-aligh: center;
}
